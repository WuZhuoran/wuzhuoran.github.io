Contribution Guide Docs Markdown
Quick Reference

Markdown is the format for all the notes in MelissaNotes. If you want to contribute to this project, please be familiar with Markdown.

Contribution Guide Docs MathJax
Quick Reference

MathJax is a LaTex formula edit framework which supports Markdown. All the formula in MelissaNotes were supported and rendered by MathJax.

All the open source notes in MelissaNotes are collected by members of Melissa Notes and follow CC BY 3.0License。
